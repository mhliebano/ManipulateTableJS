Class mngTable

By Miguel Hernandez Liebano (mhliebano@gmail.com)

This simple class allows you to manage and administer the common work with an html table (without the standard thead and tbody) containing rows (tr) and columns without the html standard.

To operate all kind depends on JQUERY in its latest version, it has not been tested in smaller versions to 3.0.0, just as it has only been tested in google chrome for linux

Instance Class

The class when it is instantiated receives two parameters in its constructor, the first of which is mandatory and corresponds to the id of the table to drive, the second parameter is a Boolean that indicates whether the first row of the table is used as column header.

If the id of the object passed in parameter to the constructor is not from a TABLE type an error is thrown by the class

Methods of Table

.table()

Return a object jquery type table

.field()

Returns an array with the names of the headers in the first row, only if defined the second constructor parameter to True

.nrows()

Return a integer with the number of rows in the table

.ncols()

Return a integer with the number of columns in the table

Methods of Data extraction

.extractDataJSON()

Return a JSON object from the data in the table: (tableId->[field1:data,field2:data,...,fieldN:data]

.extractDataCELL(r as Integer,c as Integer)

Param: r index Row, c Index Column

Returns a string of the cell whose index are specified in the parameter

.extractDataROW(r as Integer)

Param: r index Row

Returns a array with the data of each the cell into row whose index are specified in the parameter

.extractDataCOL(c as Integer)

Param: c index Column

Returns a array with the data of each the cell into column whose index are specified in the parameter

Methods of Data Insertion

.putDataCELL(r as Integer,c as Integer,t as string)

Param: r index Row, c index Column, t as String

Put a text(t) into cell whose indexes are r and c specified in the parameter

.putDataROW(r as Integer,data as Array)

Param: r index Row,data array with data to put in the cells

Place each value of the array of data in each of the cells in the row whose index is the value specified in the first parameter

Methods of CELL

.getIndexCELL(e as Event)

Param: e event for example click

Return a array with two values [0]=index of row [1]=index of Column

.findDataCELL(t as String,h as ClassName)

Param: t string to find, h name of class to remark the text found [optional if no is specified the files are hidden]

Looking for an exact match between the cells of the table, if a class name specified in the second parameters matching text is marked with the characteristic of that class. otherwise the rows are hidden and only leave visible rows with matching text

Methods of ROWS

.getIndexROW(e as Event)

Param: e event for example click

Returns an integer index of the row that triggered the event

.hideROW(r as Integer)

Param: r index Row

Hidden a row with specified index

.showROW(r as Integer)

Param: r index Row

Show a row with specified index

.addROW(r as Integer, d as Array)

Param: r index Row where you inserted the next row, d array with data to put in the cells[optional]

Insert a new row in the table

.deleteROW(r as Integer)

Param: r index Row

Delete a row with specified index

Methods of COLUMNS

.getIndexCOL(e as Event)

Param: e event for example click

Returns an integer index of the col that triggered the event

.hideCOL(r as Integer)

Param: r index col

Hidden a col with specified index

.showCOL(r as Integer)

Param: r index Col

Show a col with specified index

.addCOL(r as Integer)

Param: r index Columns where you inserted the next row

Insert a new columm in the table

.deleteCOL(r as Integer)

Param: c index column

Delete a column with specified index
